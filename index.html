FTC Robotics Intakes

Roller Intake

This intake uses rollers to pick up game pieces.

  • Pros:
  • Easy to build and maintain
  • Can pick up game pieces quickly
  • Cons:
  • May not be able to pick up game pieces that are too heavy
  • Can be noisy

Claw Intake

This intake uses a claw to pick up game pieces.

  • Pros:
  • Can pick up game pieces that are too heavy for roller intakes
  • Can be more precise than roller intakes
  • Cons:
  • Can be more difficult to build and maintain
  • May not be able to pick up game pieces as quickly as roller intakes

Hybrid Intake

This intake uses a combination of rollers and a claw to pick up game pieces.

  • Pros:
  • Can pick up game pieces quickly and precisely
  • Can handle game pieces of different weights and sizes
  • Cons:
  • Can be more difficult to build and maintain than other intakes
  • May be more expensive than other intakes
